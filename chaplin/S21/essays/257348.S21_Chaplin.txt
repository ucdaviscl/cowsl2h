Un hombre estaba caminando cuando otras cosas empezaron caer. Esas cosas incluyen la tierra y las rocas. El hombre continuó caminar, pero le estaba confundido. Luego, el hombre vio un niño que estaba lloró. El hombre no supo que hacer. Decidió recoger el niño, pero le no quería guardar el niño por mucho tiempo. Atrás de el hombre, una mujer estaba caminando rápidamente. La mujer tiene otro niño con su. El hombre vio la mujer y le dio el niño. La mujer estaba frustrada y el hombre tomó un niño. El hombre pone el niño donde se encontró, pero este tiempo, un policía le vio. La policía no pudo permitir el hombre salir el niño allí. El hombre no puede salir el niño. Luego, el hombre dio el niño a otro hombre y corró en el otro dirección. Este hombre no supo que hacer con el niño tampoco. Este hombre dio el niño a la mujer. La mujer pensó que el primero hombre le dio el niño y ellos pelean. La policía interfirió y el primero hombre tiene el niño de nuevo. El hombre necesita encontrar el lugar por el niño. Sin embargo, el hombre encontró una carta en la ropa de el niño. La carta dijo, "Por favor, encanta este orphan niño." El hombre miró a el niño y le sonrisa. El hombre decidió guardar el niño. El hombre y el niño salimos juntos. Ahora, el niño es la responsabilidad de el hombre. Y eso, este es el fin de el cuento.
